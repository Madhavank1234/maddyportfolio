01 / AGENTIC AI

CiteMind — Agentic RAG Document Intelligence

Problem: LLMs hallucinate and can't cite sources. Enterprises can't trust AI answers over sensitive documents.

Built a fully agentic document Q&A platform that ingests PDFs, PPTX, DOCX, CSV, and TXT files and returns source-cited answers with evidence chunks — eliminating hallucination risk. Multi-agent orchestration via Model Context Protocol (MCP) enables task delegation, verification passes, and retrieval routing — significantly improving answer fidelity over single-agent baselines. The UI surfaces exact citation chunks so users can verify every claim.

Multi-format doc support (5 types) Source-cited every response MCP multi-agent accuracy lift
Python FAISS HuggingFace MCP SentenceTransformer Streamlit

02 / LOCAL AI

PrivateDoc — Zero-Shot Local RAG Engine

Problem: Most RAG systems send sensitive data to external APIs. Teams with compliance requirements are locked out.

Built a 100% local RAG system — no external API calls, no data leaving the machine. OCR-based text extraction handles scanned documents. FAISS powers fast semantic vector search. TinyLlama via Ollama runs inference entirely on-device, making this viable even on consumer hardware. Designed as a zero-setup conversational interface over any uploaded document set.

Zero external API dependency OCR for scanned doc support Runs on consumer hardware
Python FastAPI FAISS Ollama / TinyLlama OCR Streamlit

03 / COMPUTER VISION

LeafScan — Agricultural Disease Detection via CNN

Problem: Plant disease diagnosis relies on expert agronomists — scarce and expensive in rural areas.

Trained a convolutional neural network on multi-class leaf disease datasets achieving 95% classification accuracy. Applied dataset augmentation and feature extraction to handle real-world photo variance (lighting, angle, phone cameras). Deployed via a Streamlit interface enabling farmers to upload a leaf photo and get an instant diagnosis — no agronomist required.
